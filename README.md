A 2D traffic simulation built in Python using Pygame featuring autonomous cars, lane-switching, dynamic speeds, and bottleneck merge logic.

## How to Run
1. Install Pygame:
   `pip install pygame`
2. Run the code:
   `python3 simulation.py
## Recommended IDE
- **Visual Studio Code:** Highly recommended for running and editing this project.
- You can easily run the script using VS Code's intergrated terminal (`Cmd + ~` on macOS) and manage Git commits directly within the editor.
